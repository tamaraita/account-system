# account-system
Account system with go backend and a react-typescript UI

### Requirements

* NodeJS (https://nodejs.org/en/)
* GoLang (https://golang.org/)

### How to run

`make run_server`

or

`make run_api`
`make_run_ui`


* Backend runs on http://localhost:8000/
* Front end runs on http://localhost:3000/
